SkillBridge Sample Inputs and Outputs
=====================================

This file contains practical sample payloads and abbreviated responses
for core APIs in this project.

------------------------------------------------------------
1) Module 2: Extract Skills
------------------------------------------------------------
Endpoint:
POST /api/module2/extract-skills

Sample form-data input:
- resume_text:
  "Python backend developer with FastAPI, SQL, and Docker. Built REST APIs and auth middleware."
- job_description:
  "Looking for Python, FastAPI, SQL, AWS, CI/CD, and system design."
- interview_date:
  "2026-05-10"

Sample response (abbreviated):
{
  "session_id": 101,
  "interview_date": "2026-05-10",
  "days_remaining": 14,
  "matching_engine": "ai:gemini-1.5-flash-latest",
  "question_generation_status": "queued",
  "present_skills": ["Python", "FastAPI", "SQL", "Docker"],
  "lacking_skills": ["AWS", "CI/CD", "System Design"]
}

------------------------------------------------------------
Error examples
------------------------------------------------------------
- 400: Interview date format invalid
  {"detail": "Interview date must be YYYY-MM-DD."}

- 404: Study plan not created yet
  {"detail": "Study plan not found."}

- 400: Practice test attempted before plan completion
  {"detail": "Complete 100% of the study plan before taking the practice test."}

- 503: AI generation unavailable
  {"detail": "AI practice test generation is currently unavailable. ..."}
